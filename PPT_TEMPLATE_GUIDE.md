# Aureus Sigma Capital - PowerPoint Template Guide
**Transitioning to the "Premium Institutional" Aesthetic**

This guide outlines how to update your PowerPoint template from the legacy style (Bright Primary Colors + Helvetica) to the new website's **Premium Institutional** look.

---

## 1. The Shift in Strategy

| Feature | **Legacy Style** (Current) | **New Institutional Style** (Target) |
| :--- | :--- | :--- |
| **Primary Color** | Bright Blue (`#00409A`) | **Deep Navy** (`#0F172A`) |
| **Accent Color** | Yellow (`#FFCA00`) | **Muted Gold** (`#D4AF37`) |
| **Typography** | Helvetica Neue (Neutral) | **Playfair Display** (Headings) + **DM Sans** (Body) |
| **Vibe** | Retail, Energetic, Tech | **Stable, Exclusive, Quantitative** |

---

## 2. Theme Colors (PowerPoint Setup)

PowerPoint uses **RGB** values. Go to **View > Slide Master > Colors > Customize Colors** and input these exact values:

### Primary Theme Colors
| PPT Slot | Color Name | Hex | **RGB Values** (Input these) | Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Background 1** | White | `#FFFFFF` | **255, 255, 255** | Content Slide Background |
| **Text 1** | ASC Navy | `#0F172A` | **15, 23, 42** | Primary Text, Title Backgrounds |
| **Background 2** | Off-White | `#F8FAFC` | **248, 250, 252** | Alternative Backgrounds |
| **Text 2** | ASC Slate | `#334155` | **51, 65, 85** | Secondary Text, Subtitles |
| **Accent 1** | **ASC Gold** | `#D4AF37` | **212, 175, 55** | **Key Highlights, Icons, Lines** |
| **Accent 2** | Slate Blue | `#64748B` | **100, 116, 139** | Charts, Secondary Data |

### Semantic Colors (For Charts/Data)
*Replace the bright Green/Red with these deeper, more professional tones:*
*   **Positive/Gain**: `#047857` (RGB: **4, 120, 87**) - *Replaces #00BD61*
*   **Negative/Loss**: `#DC2626` (RGB: **220, 38, 38**) - *Replaces #FF004C*

## 3. Typography

To match the website, you should install the Google Fonts. If that is not possible (e.g., for shared decks), use the System Alternatives.

| Role | **Primary Choice** (Website Match) | **System Alternative** (Safe Fallback) |
| :--- | :--- | :--- |
| **Headings** | **Playfair Display** (Semibold) | **Georgia** (Bold) |
| **Body Text** | **DM Sans** (Regular) | **Helvetica Neue** or **Arial** |

*Recommendation: Keep **Helvetica Neue** for body text if you prefer it, but you MUST switch Headings to **Playfair Display** (or Georgia) to achieve the "Premium" look.*

## 4. Slide Master Layouts

### A. The "Cover" Slide
*   **Background**: Solid **ASC Navy** (RGB 15, 23, 42).
*   **Logo**: White version.
*   **Title**: White, **Playfair Display**, Centered or Left-Aligned.
*   **Accent**: A thin **Gold Line** separating Title and Subtitle.
*   **Visual**: Optional subtle background image (e.g., `hcmc_skyline_1.png`) with 90% transparency.

### B. The "Section Divider" Slide
*   **Background**: **ASC Navy** (RGB 15, 23, 42).
*   **Text**: White, **Playfair Display**, Large size (e.g., 44pt).
*   **Numbering**: Large **Gold Number** (e.g., "01") to indicate the section.

### C. The "Content" Slide (Standard)
*   **Background**: White or Off-White.
*   **Title**: **ASC Navy**, **Playfair Display**, Top Left.
*   **Body**: **ASC Slate**, **DM Sans** (or Helvetica Neue).
*   **Footer**: Thin grey line with Page Number in **Gold**.

---

## 5. Chart & Graph Styling
*   **No 3D Effects**: Use flat, 2D charts.
*   **Gridlines**: Light Grey (RGB 226, 232, 240), dotted.
*   **Series Colors**:
    1.  **Series 1**: ASC Navy
    2.  **Series 2**: ASC Gold
    3.  **Series 3**: Slate Blue

---

## 6. Implementation Checklist
- [ ] Download **Playfair Display** and **DM Sans** from Google Fonts.
- [ ] Create a new blank PowerPoint.
- [ ] Set the **Theme Colors** using the RGB values above.
- [ ] Edit the **Slide Master** to apply fonts and background colors.
- [ ] Save as `.potx` (PowerPoint Template).
